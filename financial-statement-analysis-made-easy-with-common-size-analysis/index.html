Financial Statement Analysis Made Easy with Common Size Analysis


Financial statement analysis is an essential tool for understanding and evaluating a company’s financial performance. It helps investors, creditors, and other stakeholders gain insights into a company’s profitability, liquidity, solvency, and overall financial health. One popular technique used in financial statement analysis is common size analysis. In this blog post, we will explore how common size analysis can make financial statement analysis easier and more effective.

What is Common Size Analysis?

Common size analysis, also known as vertical analysis, is a technique used to analyze financial statements by expressing each item as a percentage of a base value. This allows for a standardized comparison of financial data across different companies, industries, or time periods. By converting dollar amounts into percentages, common size analysis eliminates the impact of size differences and provides a clearer understanding of the composition and trends within a company’s financial statements.

How to Perform Common Size Analysis?

Performing common size analysis involves the following steps:

  1. Select the Base Value: Choose a base value that represents 100% or 1. For the income statement, the base value is usually net sales or revenue, while for the balance sheet, it is typically total assets.

  2. Calculate the Percentages: Divide each line item by the base value and multiply by 100 to obtain the percentage. This will help you understand the proportion of each item relative to the base value.

  3. Analyze the Results: Once you have calculated the percentages for each line item, you can analyze the trends and patterns within the financial statements. Look for significant variations or deviations from the industry average or the company’s historical data.

Benefits of Common Size Analysis

Common size analysis offers several benefits that make financial statement analysis easier and more insightful:

  1. Standardized Comparison: By converting financial data into percentages, common size analysis allows for a standardized comparison of companies operating in different industries or of different sizes. This makes it easier to identify trends, patterns, and outliers.

  2. Identifying Key Drivers: Common size analysis helps identify the key drivers of a company’s financial performance. By comparing the percentage changes in different line items over time, you can pinpoint the areas that are driving growth or posing risks.

  3. Comparing Companies: Common size analysis enables you to compare companies within the same industry. By analyzing the common size income statements and balance sheets of competitors, you can identify areas where your company may be performing better or worse.

  4. Forecasting and Trend Analysis: Common size analysis allows for better forecasting and trend analysis. By observing the changes in percentages over time, you can identify emerging trends and make more accurate predictions about a company’s future performance.

Example of Common Size Analysis

Let’s take a hypothetical example to illustrate common size analysis. Suppose we have two companies, Company A and Company B, operating in the same industry. We want to compare their income statements using common size analysis:

|                  | Company A (%) | Company B (%) |
|------------------|---------------|---------------|
| Revenue          | 100           | 100           |
| Cost of Goods Sold | 60            | 70            |
| Gross Profit     | 40            | 30            |
| Operating Expenses | 20            | 15            |
| Net Income       | 20            | 15            |

By converting the dollar amounts into percentages of revenue, we can see that Company A has a higher gross profit margin (40%) compared to Company B (30%). This indicates that Company A is more efficient in generating profits from its revenue. Similarly, we can analyze other line items to gain further insights into the financial performance of both companies.

Conclusion

Common size analysis is a powerful tool that simplifies financial statement analysis and enhances the understanding of a company’s financial performance. By converting dollar amounts into percentages, common size analysis allows for standardized comparisons, identifies key drivers, facilitates company comparisons, and aids in forecasting and trend analysis. Incorporating common size analysis into your financial statement analysis toolkit will help you make more informed investment decisions and gain a deeper understanding of a company’s financial health.
